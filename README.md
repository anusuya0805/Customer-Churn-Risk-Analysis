# Customer Churn Risk Analysis Dashboard

## Project Overview
This project analyzes customer churn patterns using historical customer data.
An interactive Power BI dashboard is built to identify high-risk customers and
key factors contributing to churn.

## Objective
- Identify customer churn trends
- Analyze churn by tenure, contract type, and services
- Segment customers into churn risk levels
- Support data-driven retention strategies

## Tools Used
- Power BI
- DAX
- CSV Dataset
- Data Visualization Techniques

## Key Insights
- Customers with shorter tenure have higher churn risk
- Month-to-month contracts show the highest churn
- Fiber optic users and electronic check payments contribute more to churn
- High-risk customers require immediate retention actions

## Files Included
- Power BI Dashboard (.pbix)
- Customer Churn Dataset (.csv)
- Project Report (.pdf/.docx)
- Dashboard Screenshot

## Conclusion
This dashboard helps organizations proactively identify churn risk and
design targeted retention strategies using data-driven insights.
